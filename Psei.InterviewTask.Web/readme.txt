Opis zadania

W kontrolerze 'Students' należy zaimplementować następujące akcje:
  - pobranie listy wszystkich studentów
  - zapis nowego studenta

Lista studentów powinna być uzupełniona o nazwę wydziału z tabeli Departments, powiązanej z tabelą Students.

Nazwa wydziału powinna być uppercased

Api można testować przez interfejs Swaggera: https://localhost:5001/swagger/index.html
